# 9click-education
“九点教育系统”，旨在通过该系统来学习 webpack 的 code-spliting 和 react 服务端渲染两大核心技术。

## server 目录
> 基于 NodeJS+mongodb+express 搭建的 9click-education后台新闻管理以及相关接口服务端

### 技术栈
1. nodeJs 6.0+
2. mongodb 3.0+
3. express
4. ejs
5. restful

### 运行：
在项目所在目录根下，执行：

```shell
npm install
npm start
```

访问：localhost:5499
后台管理： localhost:5499/9clock-manage
