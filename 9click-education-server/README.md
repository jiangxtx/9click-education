# singular-education-server
> 基于NodeJS+mongodb+express搭建的singular-education后台新闻管理以及相关接口服务端

## 技术栈
1. nodeJs 6.0+
2. mongodb 3.0+
3. express
4. ejs
5. restful

## 运行：
在项目所在目录根下，执行：

```shell
npm install
npm start
```

访问：localhost:3001
后台管理： localhost:3001/cz2r-public
